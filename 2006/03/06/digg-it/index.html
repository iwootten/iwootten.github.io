Digg It

Mar 06, 2006 in WEB
1 min read

I’ve recently, through many searches to do with AJAX, Web 2.0 and the like been perusing digg.com, that fairly new web site for geeks.Basically, it’s a social booking marking site where users “digg” those sites which they find interesting, as more users digg the same pages, they get promoted up through to the main page, or higher up through the rankings within particular categories. Its a really interesting concept, quite similar to del.icio.us, but mainly centred on techie news, which means I end up reading it forever.The best part I think though is the digg spy – which allows you to see (through a fancy ajax system) which diggs are being made right now.As an example of interesting links I’ve found, check out these insanely realistic renders of various interiors.
